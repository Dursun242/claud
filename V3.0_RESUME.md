```
  + client_id (UUID)

compte_rendus:
  + semaine (INTEGER)
  + annee (INTEGER)
  + photos (TEXT[])
  + created_by_user (UUID)
  + last_edited_by_user (UUID)
  + last_edited_at (TIMESTAMPTZ)
```

### Tables Créées
```
os_validations:
  - Tracer qui a validé, quand
  - UNIQUE(os_id, client_id)

cr_commentaires:
  - Commentaires + demandes spécifiques
  - Status: 'ouvert' ou 'resolu'
  - Types: 'commentaire' ou 'demande_specifique'

chantier_photos:
  - Métadonnées photos (dimensions, tailles)
  - Lien optionnel à CR
  - Tracer qui a uploadé, quand
```

## 🔧 Services & Composants

### Services (src/app/services/)
```javascript
osService
  ├─ getAll(), getByChantier(id), getById(id)
  ├─ upsert(os), delete(id)
  ├─ validateByClient(osId, clientId)
  └─ isValidatedByClient(osId)

crService
  ├─ getAll(), getByChantier(id), getByWeek(id, semaine, annee)
  ├─ upsert(cr, userId), delete(id)
  ├─ addPhoto(crId, photoUrl), removePhoto(crId, photoUrl)
  ├─ getComments(crId)
  ├─ addComment(crId, userId, userRole, contenu, type)
  ├─ updateCommentStatus(commentId, status)
  └─ deleteComment(commentId)

photoService
  ├─ compressImage(file, maxW, maxH, quality)
  ├─ uploadToStorage(chantierId, crId, blob, fileName)
  ├─ getPublicUrl(storagePath)
  ├─ savePhotoMetadata(...)
  ├─ getChantierPhotos(chantierId)
  ├─ getCRPhotos(crId)
  ├─ deletePhoto(storagePath, photoId)
  └─ getCompressionStats(sizeOrig, sizeOpt)
```
